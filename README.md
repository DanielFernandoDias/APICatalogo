# APICatalogo

**API Asp .NET Core** desenvolvido com o objetivo de estudar os fundamentos da WEB API. 
Como:
- Model Binding,
- Roteamento,
- Data Annotations,
- Validações Personalizadas,
- Injeção de Dependências,
- Tratamento de Erros,
- Loggins,
- Entre outros.

**Por meio dessa API, busca-se adquirir um conhecimento abrangente e prático do ecossistema do ASP.NET Core, capacitando-se a criar soluções web eficientes e de alta qualidade."
